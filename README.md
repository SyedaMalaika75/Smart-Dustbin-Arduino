# 🗑️ Smart Dustbin using Arduino

## Overview
This project presents an Arduino-based Smart Dustbin that automatically opens its lid when an object is detected using an ultrasonic sensor. It promotes hygiene by minimizing physical contact.

## Features

- Automatic lid opening
- Ultrasonic sensor-based detection
- Arduino Uno
- Hygienic touch-free operation
- Embedded systems project

## Components Used

- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Breadboard
- Jumper Wires
- Power Supply

## Applications

- Smart homes
- Hospitals
- Schools
- Public places

## Project Type

Academic Embedded Systems Project
